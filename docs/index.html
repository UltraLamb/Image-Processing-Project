Image Processing Project

Feature-Based Image Processing and Reinforcement Learning for CarRacing-v3. The project converts RGB frames into compact visual road-geometry features, then evaluates the representation with PPO and SAC.

Method at a glance

  • RGB frame from Gymnasium CarRacing-v3.
  • HSV road mask and ray/radar geometry features.
  • 16D base observation with a four-frame stack for a 64D policy input.
  • PPO and SAC MLP policies used as downstream evaluation methods.

Conservative result note

PPO is the completed 500K baseline. SAC is a partial 400K fast-result branch and is not claimed to beat PPO. The comparison is not compute-equivalent, this V2 package does not include raw-pixel CNN policy training, and it does not make a solved-status claim for CarRacing-v3.
