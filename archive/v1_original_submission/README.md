# Archived Original V1 Submission

This folder holds the **original V1 PPO-only course submission**, preserved for
history and traceability.

- This is the original PPO-only project (single algorithm, earlier evaluation
  protocol).
- It is **not** the final version of the Image Processing Project.
- The final version is now in [`../../final/`](../../final/).
- V1 results and evaluation seed sets must **not** be confused with the V2
  PPO/SAC results in [`../../final/docs/RESULT_SUMMARY.md`](../../final/docs/RESULT_SUMMARY.md).

The internal subfolder structure (report, notebooks, slides, figures, tables,
artifacts) is preserved unchanged so the V1 materials remain inspectable.

## Contents

- `report/` — original V1 report PDF and LaTeX source.
- `notebooks/` — original V1 notebook.
- `slides/` — original V1 presentation.
- `figures/` — original V1 figures.
- `tables/` — original V1 CSV tables.
- `artifacts/` — original V1 artifact zip.
